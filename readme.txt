Customer portal for admin-granted digital product downloads, licenses, release notes, documentation, and support links. No checkout required.

== Description ==

Customer Download Portal lets administrators grant WordPress users access to digital products and subscription/license information. Customers log in and use the [customer_downloads] shortcode page to access five tabs: Downloads, Licenses, Release Notes, Documentation, and Support Tickets.

This is not an e-commerce plugin. There is no cart, checkout, or payment gateway.

== Installation ==

1. Upload the plugin ZIP in WordPress Admin > Plugins > Add Plugin > Upload Plugin.
2. Activate the plugin.
3. Create a page named Customer Portal.
4. Add shortcode: [customer_downloads]
5. Go to Customer Downloads > Products to create products and versions.
6. Go to Customer Downloads > Assignments to grant products to users.
7. Go to Customer Downloads > Licenses to add subscription records.
